# 史上最简单的 MySQL 教程（十一）「列类型 之 字符串型」

列类型（数据类型）
=========
所谓的列类型，其实就是指数据类型，即对数据进行统一的分类，从系统的角度出发是为了能够使用统一的方式进行管理，更好的利用有限的空间。

在 SQL 中，将数据类型分成了三大类，分别为：**数值型、字符串型和日期时间型。**
